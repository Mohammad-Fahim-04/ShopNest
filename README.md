# ShopNest

## About

ShopNest is a full-stack e-commerce web application built using the MERN Stack. It provides users with a smooth online shopping experience, including authentication, product browsing, cart management, wishlist, and order management. An admin dashboard is also available to manage products, users, and orders.

## Features

- User Authentication using JWT
- Role-Based Access (User & Admin)
- Product Search and Filtering
- Category-Based Products
- Shopping Cart
- Order Management
- Admin Dashboard
- Sales and Revenue Analytics
- Invoice Download (PDF)
- Responsive Design

## Tech Stack

- React.js
- Redux Toolkit
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Cloudinary
- MongoDB Atlas
